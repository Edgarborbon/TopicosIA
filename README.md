### MÓDULO 1: Introducción a la Inteligencia Artificial

* **PROYECTO 1:** Proyecto de investigación de inteligencia artificial e impacto ambiental

Este proyecto explora el potencial de la Inteligencia Artificial (IA) para combatir la crisis de agua en México. A través de un análisis de aplicaciones en los sectores agrícola y urbano, la investigación demuestra cómo el uso de la IA para el riego inteligente y la detección de fugas puede optimizar el consumo y fomentar una gestión hídrica más sostenible.

### MÓDULO 2: Métodos de Búsqueda Heurísticos para optimización

* **PROYECTO 1:** Sistema de enrutamiento para tiendas de autoservicio

Este trabajo presenta una implementación del algoritmo Recocido Simulado en Python. Su objetivo es resolver un problema de optimización de rutas para minimizar el costo total.
Se implementó el algoritmo del Recocido simulado el cual es crucial para escapar de mínimos locales y encontrar soluciones eficientes en este tipo de problemas. El modelo incluye una penalidad geofráfica para asegurar asignaciones lógicas
